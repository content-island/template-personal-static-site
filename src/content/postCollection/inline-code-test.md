# Test de código inline

Este párrafo contiene código inline: `const variable = "hola"` y debería verse diferente al texto normal.

Aquí hay más ejemplos:

- Variable: `let count = 0`
- HTML: `<img src="test.jpg" alt="test" />`
- CSS: `color: red; background: blue;`
- URL: `https://example.com/api/data`

## Comparación

**Texto normal**: Este es texto normal sin formato especial.

**Código inline**: `function test() { return "hello"; }` debería tener fondo gris.

**Más código**: Aquí tenemos `<div class="container">` y también `npm install package`.

## Lista con código

1. Instalar dependencias: `npm install`
2. Ejecutar servidor: `npm run dev`
3. Abrir navegador: `http://localhost:3000`

El código `console.log("debug")` debe distinguirse del texto normal.

Link para ver la diferencia: [Documentación de Markdown](https://www.markdownguide.org/basic-syntax/#code)
